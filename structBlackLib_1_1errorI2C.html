Struct BlackLib::errorI2C

This struct holds I2C errors and includes pointer of errorCore struct.

Public Members

errorCore * coreErrors
This struct initializes at constructor in BlackI2C class. Its value can set with function in BlackI2C class.
bool openError
Its value can change, when opening i2c's tty file, at function in BlackI2C class.
bool closeError
Its value can change, when closing i2c's tty file, at function in BlackI2C class.
bool setSlaveError
Its value can change, when opening TTY file, doing i2c write/read operations and assigning new device address value at functions in BlackI2C class.
bool readError
Its value can change, when reading i2c data, at functions in BlackI2C class.
bool writeError
Its value can change, when writing i2c data, at functions in BlackI2C class.

Public Functions

errorI2C ()
This function clears all flags and initializes errorCore struct.
errorI2C (errorCore *base)
This function clears all flags and assigns input parameter to coreErrors variable.
